# visitor

Projede .Net Core ve React kullanılmıştır. Görsel Bileşen Kütüphanesi olarak Ant designer tercih edilmiştir. Veritabanı olarak mssql ve entity
framework kullanılmıştır.

Proje single-page olarak çalışmaktadır.

Veri Tabanında bir adet visitor(ziyaretçi) tablosu oluşturulmuştur.
